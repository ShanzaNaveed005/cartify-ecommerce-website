# 🛒 Cartify - E-Commerce Website (PHP)

## 📌 Description

Cartify is a mini e-commerce web application developed using PHP, MySQL, and JavaScript.
It allows users to browse products, manage carts, place orders, and receive email confirmations.

---

## 🚀 Features

* User authentication (Login/Signup)
* Product browsing
* Add to cart functionality
* Order placement system
* Email confirmation using PHPMailer
* Admin panel for managing products and orders

---

## 🛠 Technologies Used

### 💻 Backend

* PHP

### 🌐 Frontend

* HTML
* CSS
* JavaScript

### 🗄 Database

* MySQL

### 📧 Email Integration

* PHPMailer

---

## 🧩 Modules

* User Module
* Admin Module
* Cart System

---

## ▶️ How to Run

1. Install XAMPP
2. Move project to `htdocs`
3. Import `database.sql` into phpMyAdmin
4. Start Apache & MySQL
5. Open browser:

   ```
   http://localhost/cartify/
   ```
